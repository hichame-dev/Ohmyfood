# Ohmyfood

## Description

Ohmyfood est une plateforme permettant aux utilisateurs de commander des plats gastronomiques directement en ligne.

## Technologies utilisées

- **HTML5**
- **CSS3 (SASS)**
- **Flexbox / Grid**
- **Git & GitHub**

## Installation

Pour cloner et exécuter ce projet localement :

```bash
git clone https://github.com/hichame-dev/Ohmyfood.git
cd Ohmyfood
```
