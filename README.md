# Ohmyfood  
Projet OpenClassrooms - Réservation de restaurants gastronomiques.

## Description
Ohmyfood est une plateforme permettant aux utilisateurs de commander des plats gastronomiques directement en ligne. Ce projet a été réalisé dans le cadre du parcours **Développeur Web** chez OpenClassrooms.

## Technologies utilisées
- **HTML5**  
- **CSS3 (SASS)**  
- **Flexbox / Grid**  
- **Git & GitHub**
