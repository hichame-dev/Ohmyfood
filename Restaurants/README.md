# Projet Ohmyfood

Un site de présentation de restaurants gastronomiques avec une interface utilisateur moderne, conçu pour une expérience **mobile first**.

## 🛠️ Technologies utilisées

- HTML5
- SCSS (CSS)
- Git & GitHub

## 📱 Fonctionnalités

- Page d'accueil avec une liste de restaurants.
- Pages individuelles pour chaque restaurant avec un menu détaillé.
- Design **responsive** adapté aux appareils mobiles.
- Animations au survol des cartes de restaurants.
